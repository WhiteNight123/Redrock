### level2 迷上买彩票的叶圣豪姐姐

**前言：**最近叶圣豪姐姐迷上了买彩票，从100个数字（1-100）里面随机选择六个数，他希望聪明的你能帮他算一下中奖的概率。

**描述：**一般情况下，如果从n个数字中抽取k个数字，就会有
$$
C_n^k = \frac{n\times(n-1)\times(n-2)\times\cdots\times(n-k+1)}{1\times2\times3\times4\times\cdots\times k}
$$
种可能，因此最终中奖的概率就是
$$
\frac{1}{C_n^k}
$$
那么我们该如何计算这个值呢？其实可以将这个公式进一步转换一下，再**利用for循环**中的`i`即可计算了：
$$
\frac{(n-1+1)\times(n-2+1)\times(n-3+1)\times\cdots\times(n-i+1)}{1\times2\times3\times4\times\cdots\times i}
$$
**要求：**输入两个数字n和k（都为正整数），并计算从n个数字中抽取k个数字进行抽奖，输出最终的中奖概率。

**进阶PLUS(量力而行）：**在写完level2作业后，当你输入比较大的n值时（例如n为100，k为10时），最终输出的概率会为0.0000...，这是因为
$$
C_{1000}^{10}
$$
的值非常大，其倒数则非常小（大约为5.776904乘以十的负14次方）。而要解决这个问题，则可以利用java.math函数里面提供的**BigDecimal类**，正如它的名字所示，它是用来存储较大位数的小数的，你可以使用`BigDecimal.valueOf(1)`来创建一个数字1，使用它的`.multipy()`方法进行乘法运算，使用它的`.divide()`方法进行除法运算。（提示：divide()函数里需要提供三个参数，第一个参数为除数，第二个参数为保留位数，第三个参数为保留模式`RoundingMode`即是否为四舍五入，**具体的可以按住<kbd>ctrl</kbd>并点击相应的类或方法即可查看其对应的源代码**）。

### level3 帮助王中泰姐姐

**前言：**新学期来了，王中泰姐姐想要写一个学生管理系统来管理报名红岩网校的学员信息，但是要写出一个完整的学生管理系统需要花费不少时间，于是他想要聪明的你利用Java里面的**数组**来实现简单的学生管理系统，只需要可以增删查改即可。

**描述**：你只需要存储学生的**学号和姓名**这两个信息，例如：一个学生可以使用一个一维数组来表示：

```java
String[] student = { "2021123456", "邱天" };
```

那么一组学生的信息就可以使用**二维数组**来表示：

```java
String[][] students = { 
    { "2021123456", "邱天" }, 
    { "2021234567", "王中泰" } 
};
```

这样你就可以成功利用数组创建一个学生信息数据库(伪)了！

接下来让我们依次分析数组的增删查改该如何实现：

首先是如何增加学生信息，先使用`Scanner`分别读取学生的学号和姓名，然后遍历储存学生信息的二维数组，**如果当前遍历的数组元素的值为空**，则将输入的学号和姓名构成一个新的数组并赋值给当前遍历的二维数组。

那么该如何判断当前遍历的二维数组元素的值是否为空呢？请注意，假如你定义了一个m行2列的二维数组s，**不能直接判断其第i个值是否等于null**，因为**`s[0] 不等于 null`，`s[0][0] 才等于 null`**（这里涉及到对象的知识，现在可以先不用搞懂为什么）。

```java
// 创建一个m行2列的二维数组，其中m为自己输入的一个数字
String[][] s = new String[m][2];
```

所以当你判断二维数组中第i个元素是否为空时可以这样来写：

```java
for (int i = 0; i < s.length; i++) {
    if (s[i][0] != null && s[i][1] != null) {
        // 在这里对s[i]重新赋值
    }
}
```

接着是删除学生信息的操作，其实大体上和增加学生信息的操作一致，也是先遍历二维数组再判空，**如果不为空且当前遍历数组的学号和输入的查询学号一致的话**，则将当前遍历的数组置空（即赋值为**两个元素都为null的二维字符串数组**）。

在修改学生学号操作中，我们还需要输入要修改的学生信息（包括学号和姓名），在搜索操作中只需要遍历数组并将学号相匹配的数组输出即可（将数组转换为字符串时可以直接使用`Arrays.toString()`）。

最后当你写完增删查改的操作之后可以试着完善一下程序，例如：将增删查改的四个操作分别放在四个函数中，在主函数里通过`switch()`语句判断要执行哪个操作，并在`case`中调用相应的函数；还可以试着对增删查改四个操作的最终结果成功与否进行输出说明；如果你仍有余力，还可以试着结合前两个Level的作业对程序进行一些完善哦。

**进阶PLUS（量力而行，三思而后行）：**使用**冒泡排序**按照**学号大小依次从小到大**打印出所有学生信息。注意这里需要判断当前遍历的储存学生信息的数组是否为空哦，判空方法在上面已经提到了。你可以使用`Integer.parseInt()`方法将字符串转换为整型数字。
